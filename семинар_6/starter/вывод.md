## Итоговый отчет 

Валидатор встроен в run_pwc().

Сигнатура:
VALID_TOOLS = {"get_fx_rate", "get_key_rate", "get_inflation", "calculate"}

def validate_plan(plan: Plan) -> list[str]:
    errors = []
    for sq in plan.subquestions:
        fake = set(sq.expected_tools) - VALID_TOOLS
        if fake:
            errors.append(
                f"Подвопрос {sq.id} использует несуществующие инструменты: {sorted(fake)}"
            )
    return errors

PWC оправдан, когда задача может быть явно декомпозирована на отдельные подзадачи, также, когда можно настроить параллельность выполнения некоторых задач. Если существует БОЛЬШОЕ количество разных инструментов и модель склонна к галюцинациям на них. Кроме того, использование pwc может быть полезно в ситуациях, когда нужна предсказуемая структура вывода.
