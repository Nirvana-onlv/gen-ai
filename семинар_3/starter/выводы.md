# Итоговый вывод по работе

Мной был выбран датасет с kaggle с отзывами на мобильные телефоны.
Модель обрабатывала первые 51 отзыв из датасета. Стоит также учесть, что в датасете есть выбросы и неточности, чтобы возникало приближение к реальным данным. Я не стал удалять их.

--Первая часть (IE)--

В ходе обработки данных проблем у модели не возникло. Все 51 отзыв были обработаны корректно. ValueError не возникало. По некоторым отзывам модель возвращала существенное количество полей пустыми в т.ч. поле issues. С учетом того, что некоторые отзывы были короткими и могли не содержать достаточнной информации, думаю, это можно считать валидным.

--Вторая часть (аспектный анализ)--

Модель также позитивно справилась с этим заданием - все 51 отзыв были обработаны. Ghost-цитат 9 из 248, что составляет (3.6%), следовательно порог пройден.
